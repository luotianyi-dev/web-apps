重要提示

为保护您的隐私，不建议在「天依卡申请系统」中填写您的真实姓名、真实生日和其他真实信息。Tianyi Network 将您填写的信息视为虚构信息，所施加的保护措施等级低于个人信息 (PII) 保护等级。若您在「天依卡申请系统」中填写的信息泄露，Tianyi Network 不承担任何责任。

请注意保管您的「天依卡签名密钥」。「天依卡签名密钥」是一个 40 位字母 A-F 和数字组成的号码（使用 SHA-1 算法生成），用于您在日后下载「天依卡」时验证您的身份。任何知道「天依卡签名密钥」的人都可以下载对应的「天依卡」。

请注意防范以 Tianyi Network 或其他名义向您您索要真实身份信息、财务信息或其他敏感信息的诈骗。

「天依卡申请系统」相关内容纯属虚构，「天依卡」不是任何国家或地区的有效身份证件或旅行证件，如有雷同纯属巧合。

建议使用最新版的电脑端 Chrome 或 Firefox 浏览器。此系统不支持 Internet Explorer 和 UWP 版本的 Edge 浏览器。

编码规范

「天依卡申请系统」使用标准中文电码标识姓名，您需要确保您姓名中的字符被《标准中文电码本》收录。若您的姓名中包含非《标准中文电码本》收录的字符，该字符对应电码将表示为 0000。虽然系统允许，但是原则上您不应该生成姓、名所有字符对应电码均为 0000 的「天依卡」。

根据国际民航组织相关规定，非二元性别者的「性别」字段应填写为 X。

「天依卡申请系统」生成的「天依卡」符合 ICAO Doc 9303 规范。证件设有机读码（MRZ）区域，位于「天依卡」下方，并可根据 ISO 规范进行校验。
