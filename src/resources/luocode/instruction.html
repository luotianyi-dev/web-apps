1. 此算法并不是密码学意义上的加密算法，而是编码（encode），因为其算法公开且使用公共密码本，任何能访问此网页的人都能够随意解密密文。 请勿用于需要真正加密的场景，因为任何人都将可以直接解密您的密文。
  2. 如果你需要解密，请把密文输入在下面的文本框中。
    加密时，始终将需要加密的内容输入在上面的文本框中，点击「↓ 加密」按钮，将会在下面的文本框中显示密文。
    解密时，始终将需要解密的内容输入在下面的文本框中，点击「解密 ↑」按钮，将会在上面的文本框中显示明文。
    将需要加密的内容输入在下面后直接点「↓ 加密」，或将需要解密的内容输入在上面后直接点「解密」按钮，将导致你输入的内容被覆盖。
    我们不会保存任何你加密、解密的历史记录，数据没有了就是没有了。
  3. 您加密的内容不会发送到服务器，所有加密解密操作均在您的浏览器中完成。您可以按下 F12 打开开发者工具，切换到 Network 标签页，然后点击加密或解密按钮，查看请求列表，来确认这一点。
  4. 请不要使用此工具加密违反您所在国家或地区法律法规的内容及敏感信息。我们会对加密内容中的敏感词进行检查，包含敏感词的加密内容将会被拒绝。 我们不会将您的加密内容发送到服务器，敏感词检查是从服务器下载敏感词库后在本地运行的。 其原理为从服务器下载包含敏感词 SHA-1 值的列表，并对您输入内容进行分词后逐个进行不可逆的 SHA-1 计算，然后与列表中的 SHA-1 值进行比对。 我们也不会记录试图加密敏感词的请求或追踪试图加密敏感词的用户身份。 敏感词屏蔽功能需要特定的浏览器 API 支持，若您的浏览器不支持，则敏感词检查功能将关闭，您依然可以使用此工具。
